# 销售链接模板 · DD Deep Design 选品中心

发给 B 端客户前，复制对应链接（可按项目替换 `invite`）。

基础地址：`https://zhouxianliang73.github.io/dd-design/1.html`

| 来源独立站 | channel | 示例链接 |
|------------|---------|----------|
| Outdoor Living（户外阳台整体） | `outdoor-living` | `https://zhouxianliang73.github.io/dd-design/1.html?channel=outdoor-living&invite=客户简称` |
| Outdoor Kitchen（户外厨房） | `outdoor-kitchen` | `https://zhouxianliang73.github.io/dd-design/1.html?channel=outdoor-kitchen&invite=客户简称` |
| 法式家具（俄罗斯） | `french-ru` | `https://zhouxianliang73.github.io/dd-design/1.html?channel=french-ru&invite=客户简称` |
| 俄罗斯活动家具 | `event-ru` | `https://zhouxianliang73.github.io/dd-design/1.html?channel=event-ru&invite=客户简称` |
| 定制 / 销售直链 | `custom` | `https://zhouxianliang73.github.io/dd-design/1.html?channel=custom&invite=客户简称` |

已有项目可加：`&project=项目id`

独立站「进入协作」按钮应指向带 **对应 channel** 的链接，不要链到无参数首页。
