图片内容导入Excel

用一张标准 Excel 维护设计师头像、商品图、案例封面。本地图片与 Excel 一并上传后，自动写入 JSON 并同步到小程序。

使用步骤

  1. 下载模板：运行 npm run content:template 生成 content/dd-content-template.xlsx
  2. 填写 Excel：三张表「设计师」「商品图片」「案例封面」。图片列填网址或文件名（如 zhangjun.jpg）
  3. 准备图片：本地图放到 content/images/，或与 Excel 一起在下文上传
  4. 导入：上传 Excel + 图片，点击「开始导入」；微信开发者工具重新编译即可预览
