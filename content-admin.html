三表标准：设计师 · 商品清单（外网接入标准，与 catalog.json 一致）· 案例封面。 外网 Excel 先按「商品清单」列清洗，再导入入库。

使用步骤

  1. 下载模板：npm run content:template → content/dd-content-template.xlsx
  2. 外网文件：供应商表先对齐「商品清单」列（见「商品清单说明」页），支持常见列名别名
  3. 填写 Excel：设计师 / 商品清单 / 案例封面；商品可新增行（新 ID）或更新已有行
  4. 准备图片：本地图放到 content/images/
  5. 导入：本页上传 → 清洗写入 catalog.json 等 → 自动 sync 小程序
